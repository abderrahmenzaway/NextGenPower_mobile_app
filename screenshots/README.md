## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (preferred) or JPG
- **Naming**: Use exact filenames listed above (lowercase, underscores)
- **Content**: Clear, well-lit, showing key features
- **Mobile**: Portrait orientation for mobile screenshots

## How to Take Screenshots

### Desktop/Web Screenshots
1. Open the application
2. Navigate to the desired screen
3. Press `Windows + Shift + S` (Windows) or `Cmd + Shift + 4` (Mac)
4. Save with the appropriate filename

### Mobile App Screenshots
1. Run the app: `flutter run`
2. Navigate to each screen
3. Take screenshots on your device/emulator
4. Name files according to the list above

### Architecture Diagrams
- Use tools like draw.io, Lucidchart, or similar
- Export as PNG with transparent background
- Show component relationships clearly
