# Screenshots Directory

Place your mobile app screenshots here with these exact filenames:

- `login_screen.png` - Login and registration interface
- `dashboard.png` - Main dashboard showing available factories  
- `my_factory.png` - Factory energy monitoring and statistics
- `trading_offers.png` - Energy trading marketplace
- `blockchain_screen.png` - Blockchain transactions
- `profile_screen.png` - User profile and settings

**Format**: PNG, Portrait orientation, 1080x1920 or similar mobile resolution
